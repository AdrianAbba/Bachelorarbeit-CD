EASILY DEPLOY YOUR APP

Meteor Up is a production quality Meteor app deployment tool.


Install with one command:

$ npm install --global mup



Getting Started View on Github


Why Meteor UP?

Ship your application with less headache and overhead.
Deploy from anywhere. Meteor Up runs on Windows, Mac, and Linux.
Deploy to anywhere. Meteor Up can run your app on any Ubuntu server you can ssh into.

Single-command Server Setup

Meteor Up can install all of it's dependencies on the servers and prepare them for deployments.

SSL Support

Meteor up can setup Let's Encrypt to create free certificates automatically. It also supports using your custom certificates.

Application Logs

View logs from the terminal, with log tailing and following supported.

Automated Rollback

Meteor Up verifies that the app sucessfully started, and rolls back to the previous version if it failed.
